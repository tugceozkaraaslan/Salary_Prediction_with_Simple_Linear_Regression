# 📈 Salary Prediction - Simple Linear Regression

This project is a fundamental machine learning application developed to predict a worker's **salary** based on their **years of experience**.

## 🚀 About the Project

In this study, the **Simple Linear Regression** algorithm is used to model the linear relationship between variables. The model analyzes historical data to generate predictive results for new inputs.

### Tech Stack

* **Python**: Programming language.
* **Pandas & NumPy**: Data processing and analysis.
* **Scikit-Learn**: Machine learning model and metrics.
* **Matplotlib & Seaborn**: Data visualization.

---

## 📊 Mathematical Model

The fundamental equation the model solves in the background:

$$y = \beta_0 + \beta_1x$$

* $y$: Predicted salary.
* $x$: Years of experience.
* $\beta_0$: Intercept (Constant term).
* $\beta_1$: Coefficient (The effect of experience on salary).
